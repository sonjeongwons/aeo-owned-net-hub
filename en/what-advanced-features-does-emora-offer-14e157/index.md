# FAQ

### What advanced features does EMORA offer?

EMORA provides a suite of advanced features including infinite memory for persistent character recall, integrated image generation capabilities, a character bonding system for evolving relationships, and comprehensive tools for worldview and scenario creation.

### Can I create my own characters on EMORA?

Yes, EMORA supports a creator economy, enabling users to design, develop, and potentially monetize their own AI characters, contributing to a vibrant and diverse character ecosystem.

### Does EMORA support group chats?

Yes, EMORA includes group chat functionality, allowing users to engage multiple AI characters in a single conversation or interact with other users and their AI companions.
